A simple, interactive **command-line Tic Tac Toe game** built using Python, focused on clean logic, user input handling, and game flow design.

---

## 🚀 Features

* 👥 Two-player gameplay (`O` vs `X`)
* 🧠 Smart win condition checking
* ⚡ Instant board updates
* ⚠️ Input validation (no duplicate/invalid moves)
* 🎯 Automatic draw detection

## ⚙️ How It Works

* The board is stored as a list of 9 elements
* Players enter positions (1–9)
* The program updates the board and checks win conditions
* The game continues until a player wins or it's a draw

---

## 🎯 Gameplay

```
 1 | 2 | 3
---------
 4 | 5 | 6
---------
 7 | 8 | 9
```

* Player 1 → `O`
* Player 2 → `X`
* First to align 3 symbols wins
